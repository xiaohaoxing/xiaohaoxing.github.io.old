postgreSQL index 翻译

##9.26.8. 索引维护函数
下表展示了索引维护任务需要用到的函数。恢复的过程中不会执行这些函数。仅限超级用户和索引的owner使用这些函数。
表9.87 索引维护函数
|函数名|返回值类型|描述信息|
|————-|——–|——————————-|
|brin_summarize_new_values(index regclass)|integer|归纳尚未归纳的页范围|
|brin_summarize_range(index regclass, blockNumber bigint|integer|归纳覆盖了给定块且尚未归纳的页范围|
|brin_desummarize_range(index regclass, blockNumber bigint|integer|去归纳(de-summarize)覆盖给定块且已归纳的页范围|
|gin_clean_pending_list(index regclass)|bigint|将GIN偏移列表实体移入主索引结构|
